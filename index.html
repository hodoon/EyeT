Project: EyeT

AI 기반 사시 유형 판별 및 교정 훈련 솔루션

1. 문제 정의 (Problem)

소아 사시는 조기에 발견하지 못하면 약시나 시력 저하로 이어질 수 있습니다. 하지만 기존 병원 기반 진단은 정기 방문이 필요하고 가정에서 자가 진단이 어렵다는 문제가 있었습니다.

본 프로젝트의 목표는 가정에서도 쉽고 정확하게 사시 유형을 판별하고, 맞춤형 교정 훈련을 제공하여 조기 교정을 지원하는 AI 기반 솔루션을 개발하는 것입니다.

2. 아키텍처 (Architecture)

카메라 입력 → MediaPipe Iris → YOLOv5 + RoboFlow → 교정 게임 엔진 구조로 설계하였습니다.

  • MediaPipe Iris: 실시간 시선 추적, 동공 좌표 3D 추출
  • YOLOv5 + RoboFlow: 사시 유형(상/하/좌/우) 분류를 위해 학습된 커스텀 객체 탐지 모델
  • 게임 엔진 (Unity / React Native 연동): 맞춤형 교정 훈련 제공
  • 데이터 관리: Roboflow를 통한 이미지 라벨링 및 데이터 증강 적용
  • 배포 환경: PC 및 모바일 카메라 기반 실시간 동작 지원, 최소한의 하드웨어 요구

3. 핵심 기능 (Features)

실시간 동공 시선 추적

MediaPipe Iris 모델을 활용해 양쪽 눈의 홍채 중심 좌표와 동공 위치를 실시간으로 추출하여 시선 이동을 X/Y 좌표로 변환하여 사시 각도를 계산, OpenCV 기반 필터링으로 조명/각도 변화에도 안정적 추적 가능

사시 유형 분류

YOLOv5 모델을 RoboFlow로 라벨링, 증강한 약 1,500장의 학습 데이터로 학습하여 상사시, 하사시, 좌사시, 우사시를 정확히 분류한 후 MediaPipe Iris로 추출한 눈 영역과 동공 좌표를 입력으로 사용하여 맞춤형 교정 훈련에 연동

맞춤형 교정 게임

사시 유형에 맞춰 게임 내 목표물 위치와 난이도를 조정하여 사용자에게 맞춤형 훈련을 제공

4. 나의 역할 및 기여 (My Role)

MediaPipe Iris 기반 실시간 동공 추적 모듈 설계 및 구현, YOLOv5 모델 학습용 데이터셋 구축, RoboFlow 활용 데이터 라벨링 및 증강, 사시 유형 분류 모델 학습, 정확도 검증 및 튜닝, 교정 게임 연동 로직 설계 및 Unity/React 통합을 담당했습니다.

5. 결과 및 성과 (Results)

  • YOLOv5 모델 정확도: 상사시(93.4%), 하사시(93.2%), 좌사시(92.8%), 우사시(92.6%)
  • MediaPipe Iris 기반 실시간 시선 추적: CPU 환경에서도 25~30 FPS 실현
  • 맞춤형 교정 게임 연동 성공, 사용자 테스트에서 흥미도 및 지속 참여율 증가 확인

6. 회고 (Retrospective)

배운 점 (What I Learned)

  • YOLOv5 객체 탐지와 MediaPipe Iris 기반 시선 추적을 결합하여 실시간 AI 서비스 설계를 경험.
  • Roboflow를 활용한 데이터 증강과 라벨링 과정의 중요성을 이해.

개선점 (Improvements)

  • 모바일 환경 최적화를 위해 YOLOv5 경량화 또는 TensorRT 변환 필요.
  • 장기간 학습 데이터 축적을 통한 사시 분류 정확도 향상 및 추가 안질환 확장 가능.
  • 게임 피드백 알고리즘 고도화, 시선 정확도 기반 난이도 자동 조정 기능 개선 가능.
